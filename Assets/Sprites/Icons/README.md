# Store Icon Assets

## Required Files

### AppIcon_512.png
- **Size**: 512×512 px
- **Format**: PNG, no alpha (solid background)
- **Design brief**:
  - Background: deep dark navy (#0A0A14)
  - Center: "VD" monogram or vault symbol in gold (#D4A800)
  - Subtle speed lines / lightning radiating from center
  - No rounded corners (Google Play applies mask automatically)
  - No text except logo mark

- **Scenario.gg prompt**:
  ```
  App icon for a mobile runner game called "Vault Dash". 
  Dark navy background, gold vault door symbol in center 
  with speed lines, neon blue accent glow, minimal design, 
  512x512, no text, game icon style.
  ```

### FeatureGraphic_1024x500.png
- **Size**: 1024×500 px
- **Format**: PNG, no alpha
- **Design brief**:
  - Background: isometric tunnel corridor (dark + neon)
  - Characters: 3-4 Vault Dash characters running in parallax
  - Foreground text: "VAULT DASH" (large, bold, white + gold)
  - Subtext: "Real-time 1v1 Runner" (smaller, grey)
  - Trophy/rank icons on right side
  - Tagline: "Outsprint rivals. Claim your rank."

- **Scenario.gg prompt**:
  ```
  Feature banner for mobile game "Vault Dash". 
  1024x500px horizontal. Three characters running through 
  an isometric neon corridor. Text: "VAULT DASH - Real-time PvP Runner". 
  Dark navy theme, gold accents, cinematic lighting.
  ```
